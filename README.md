# Xử lý ảnh và Thị giác máy tính (Computer Vision)

Dự án thực hành cho môn học **Xử lý ảnh và thị giác máy tính**. Dự án bao gồm các bài tập về xử lý ảnh sử dụng Python, OpenCV, và Jupyter Notebook.

## Cấu trúc thư mục
- `Lab1.ipynb`: Nội dung bài thực hành Lab 1 (Xử lý cơ bản, đọc/ghi ảnh, chuyển đổi không gian màu...).
- `in.jpg`: Ảnh mẫu đầu vào phục vụ cho các bài tập.
- `requirements.txt`: Danh sách các thư viện Python cần thiết.
- `.venv/`: Môi trường ảo (Virtual Environment) để cô lập các thư viện của dự án.
- `.gitignore`: Cấu hình Git để loại bỏ các file rác và thư mục môi trường ảo khi đẩy lên GitHub.

## Các thư viện chính sử dụng
- **OpenCV (opencv-python):** Thư viện xử lý ảnh chính.
- **Matplotlib:** Hiển thị ảnh và biểu đồ.
- **NumPy:** Xử lý mảng đa chiều hiệu quả.
- **Pillow:** Thư viện xử lý ảnh bổ trợ.

## Cài đặt & Sử dụng

### 1. Chuẩn bị môi trường (Khuyên dùng)
Nếu bạn chưa có môi trường ảo, hãy tạo mới:
```powershell
python -m venv .venv
```

### 2. Kích hoạt môi trường ảo
Trên Windows:
```powershell
.\.venv\Scripts\activate
```

### 3. Cài đặt thư viện
Cài đặt tất cả các thư viện cần thiết từ file `requirements.txt`:
```powershell
pip install -r requirements.txt
```

### 4. Chạy dự án
Mở Jupyter Notebook để xem và thực thi mã nguồn:
```powershell
jupyter notebook
```
Sau đó, chọn file `Lab1.ipynb` từ trình duyệt hiện ra.
